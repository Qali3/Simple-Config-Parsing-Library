# Simple-Config-Parsing-Library
SCPL (Simple Config Parsing Library) - is a library for a simple, fast, zero-copy and flexible parsing of scpl-format in C-lanugage.

SCPL syntax example:
```scpl
	theme = [
		number = 4,
		string = 'foo',
		arrays = [5, 'f'],
		tables = [
			something = 'a'
		]
		// Comment
		/* Comment Block */
	]
```
